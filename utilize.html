Objetivo

O VerdiCode tem com foco a Automação de práticas sustentáveis, como gerenciamento de energia em residências inteligentes, monitoramento ambiental ou otimização de recursos em fazendas urbanas. Projetada para usuários não-especialistas, como ambientalistas ou engenheiros de IoT, facilitando scripts rápidos para reduzir desperdício.

Requisitos

  • Interpretador Lua 5.4 ou superior (Baixe Aqui)
    •
  • Bibliotecas Lua opcionais
  • IDE como VS Code com extensão Lua
  • Para IoT: Compatível com plataformas como ESP32 ou Raspberry Pi.
  • Custo: Gratuito. Não requer licenças ou hardware especial além do Lua padrão.
